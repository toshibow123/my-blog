# 📊 Google Analytics 測定IDの取得方法

## 方法1: 管理画面から取得（標準的な方法）

1. **左側のメニューから「管理」（⚙️ 歯車アイコン）をクリック**
2. **プロパティ列（真ん中の列）を確認**
3. 以下のいずれかを探してください：
   - **「データストリーム」** → クリック → 「Web」ストリームを選択
   - **「データ収集」** → 「データストリーム」→ 「Web」ストリームを選択
   - **「ストリーム」** → 「Web」ストリームを選択

4. **Webストリームをクリック**すると、測定ID（`G-XXXXXXXXXX`）が表示されます

## 方法2: プロパティ設定から確認

1. **左側のメニューから「管理」をクリック**
2. **プロパティ列の「プロパティ設定」をクリック**
3. ページ上部に「測定ID」が表示されている場合があります

## 方法3: 既存のウェブストリームから確認

もし既にウェブストリームが設定されている場合：

1. **ホーム画面の上部**に「www.toshiboh.com」と表示されている部分を確認
2. **「管理」→「データストリーム」**（または「ストリーム」）
3. 既存の「Web」ストリームをクリック
4. 測定IDが表示されます

## 方法4: URLから直接確認

Google AnalyticsのURLが以下のような場合：
```
https://analytics.google.com/analytics/web/#/p123456789/streams/...
```

この場合、既にストリームが設定されているので、そのページで測定IDを確認できます。

## 測定IDの形式

測定IDは以下の形式です：
- `G-XXXXXXXXXX`（G-で始まる10文字の英数字）

## 見つからない場合

もし上記の方法で見つからない場合は：

1. **新しいウェブストリームを作成**
   - 「管理」→「データストリーム」→「ストリームを追加」→「ウェブ」
   - URL: `https://www.toshiboh.com`
   - ストリーム名: `Web` または `www.toshiboh.com`
   - 「ストリームを作成」をクリック
   - 作成後、測定IDが表示されます

2. **Google Analyticsのバージョンを確認**
   - GA4（Google Analytics 4）を使用していることを確認
   - 古いUniversal Analyticsの場合は、GA4に移行が必要です

## スクリーンショットで確認したい場合

Google Analyticsの画面で：
- **左側のメニュー**を確認
- **「管理」**（⚙️アイコン）をクリック
- **プロパティ列**（真ん中の列）の項目を確認

「データストリーム」が見つからない場合は、画面のスクリーンショットを共有していただければ、より具体的に案内できます。
