# 📈 ブログ集客力を高める方法

## 実装済みの機能

### 1. SNSリンク
- **プロフィールカード**: Twitter、GitHub、LinkedInのリンクを追加
- **フッター**: SNSリンクとRSSフィードアイコンを追加
- **設定方法**: `components/SocialLinks.tsx` のURLを実際のものに置き換えてください

### 2. RSSフィード
- **フッター**: RSSフィードアイコンを追加（`/feed.xml`）
- 読者が記事を購読しやすくなります

### 3. メールマガジン登録フォーム
- **サイドバー**: メールマガジン登録フォームを追加
- 現在はローカルストレージに保存（本番環境ではMailchimp、ConvertKitなどと連携）

## 設定が必要な項目

### SocialLinks.tsx
以下のURLを実際のものに置き換えてください：

```tsx
{
  name: "Twitter",
  url: "https://twitter.com/your_username", // ← 実際のTwitter URL
},
{
  name: "GitHub",
  url: "https://github.com/toshibow123", // ← 実際のGitHub URL
},
{
  name: "LinkedIn",
  url: "https://www.linkedin.com/in/your-profile", // ← 実際のLinkedIn URL
}
```

### メールマガジンサービスとの連携
`components/NewsletterForm.tsx` を編集して、MailchimpやConvertKitのAPIと連携してください。

## 集客のコツ

1. **継続的な投稿**: 週1回以上、定期的に投稿する
2. **SNSでの拡散**: 記事を投稿したら、Twitterなどでシェア

4. **価値のあるコンテンツ**: 読者の悩みを解決する記事を書く
